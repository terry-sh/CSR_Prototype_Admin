# CSR Server

## Dependencies

Runtime: Python 3.9+
Toolchain: pipenv

```bash
pip3 install --user pipenv
```

PIP Packages:
- Django
- djangorestframework
- djangorestframework-simplejwt
- djangorestframework-dataclasses
- django-filter

## Start up

```bash
# pipenv --python 3.9

pipenv install
```
